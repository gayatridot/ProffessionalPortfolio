## 🚀 Getting Started

### Prerequisites

No special dependencies required! This is a static website.

### How to View

1. **Option 1: Local File**
   - Navigate to the project folder
   - Double-click `index.html` to open in your default browser

2. **Option 2: Local Server** (Recommended for better performance)
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Or using Node.js (http-server)
   npx http-server
   ```
   Then visit `http://localhost:8000` in your browser

3. **Option 3: Deploy Online**
   - Upload files to GitHub Pages
   - Use Netlify or Vercel for hosting
   - Share the live link with recruiters
