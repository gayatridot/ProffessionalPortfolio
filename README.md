# Gayatri Ghogare | Premium Portfolio

A world-class, recruiter-focused, single-page portfolio website designed for high-impact professional presentation. This portfolio positions Gayatri as a **Frontend Developer with Android Development Experience and Growing AI Expertise.**

## ✨ Key Features
- **Premium Aesthetics**: Minimalist, high-performance design inspired by the top 0.1% developer portfolios.
- **Dynamic Animations**: Powered by **GSAP** (GreenSock Animation Platform) for smooth scroll-driven effects and section entrances.
- **Dark Mode Support**: Sleek, eye-friendly dark mode with persistent state using LocalStorage.
- **Glassmorphism UI**: Modern navigation and interface elements with blur and transparency effects.
- **Responsive & Accessible**: Fully optimized for all screen sizes and built with semantic HTML for SEO and accessibility.
- **Category Filtering**: Interactive project showcase with dynamic filtering (Web, Mobile, AI).
- **Recruiter Optimized**: Clear calls-to-action, achievement-oriented experience cards, and role-specific resume downloads.

## 🛠️ Technical Arsenal
- **Core**: HTML5, Vanilla CSS3 (Custom Design System), JavaScript (ES6+).
- **Libraries**:
  - **GSAP**: Industry-standard animations.
  - **Lucide**: Clean, consistent SVG icon set.
  - **ScrollTrigger**: Advanced scroll-based interactions.
- **Deployment**: Vercel-ready architecture.
